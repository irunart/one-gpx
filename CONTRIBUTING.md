# Contributing to One GPX for Trail Running

This repository is the working group for GPX standardization for the trail running scenario.

## How to Contribute

The best way to contribute is to participate in discussions on the [GitHub issue tracker](https://github.com/irunart/one-gpx/issues).

We welcome:
-   Proposals for new features or improvements.
-   Reports of issues and inconsistencies in GPX files for trail running.
-   Feedback on existing proposals.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
